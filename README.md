# Weather Console Application

This is a simple console application that fetches and displays the current weather for a specified city.

## Project Structure

- **Program.cs**: Entry point of the application. Prompts the user for a city name and displays the weather information.
- **WeatherService.cs**: Contains the `WeatherService` class responsible for fetching weather data from an external API.
- **Models/WeatherData.cs**: Defines the `WeatherData` class, representing the structure of the weather data returned from the API.
- **WeatherConsoleApp.csproj**: Project file containing configuration settings for the application.

## Getting Started

1. Clone the repository or download the source code.
2. Open the project in your preferred C# development environment.
3. Ensure you have the necessary dependencies installed.
4. Run the application.

## Usage

When you run the application, you will be prompted to enter a city name. After entering the city, the application will fetch and display the current weather information.

## Dependencies

- .NET SDK (version required)
- Any additional libraries used for API calls (if applicable)
